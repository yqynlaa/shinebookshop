# Shine书店项目配置说明

## 1. 项目概述
这是一个基于Java Web的在线书店系统，使用JSP/Servlet技术栈。

## 2. 数据库配置

### 2.1 数据库信息
- 数据库名：`shinebookshop`
- 配置文件：`src/dbinfo.properties`
- 数据库脚本：`shinebookshop.sql`

### 2.2 配置参数
```properties
# 数据库连接URL
DB_URL=jdbc:mysql://localhost:3306/shinebookshop?useUnicode=true&characterEncoding=UTF-8&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true&useSSL=false

# 驱动类名
DB_DRIVER=com.mysql.cj.jdbc.Driver

# 用户名和密码（请根据您的MySQL配置修改）
DB_USERNAME=root
DB_PASSWORD=root

# 连接池配置
DB_MAX_ACTIVE=20
DB_INITIAL_SIZE=1
DB_MAX_WAIT=60000
DB_MIN_IDLE=1
```

## 3. IntelliJ IDEA配置步骤

### 3.1 导入项目
1. 打开IntelliJ IDEA
2. File → Open
3. 选择项目根目录：`C:\shinebookshop\shinebookshop`
4. 选择"Open as Project"

### 3.2 项目设置
1. **项目结构**：
   - Project SDK：Java 1.8
   - Project Language Level：8

2. **模块设置**：
   - Source Folders：`src`
   - Web Resource Directories：`WebContent`
   - Web.xml：`WebContent/WEB-INF/web.xml`

3. **库依赖**：
   项目已配置以下JAR包：
   - MySQL Connector：mysql-connector-java-5.1.10-bin.jar
   - JSON处理：json-lib-2.3-jdk15.jar
   - Apache Commons系列库
   - JSTL：jstl-1.2.jar
   - Servlet API

### 3.3 Web应用配置
- **部署描述符**：WebContent/WEB-INF/web.xml
- **Web资源根目录**：WebContent
- **默认页面**：index.html, index.jsp

## 4. 运行环境要求
- **JDK版本**：Java 8+
- **Web服务器**：Apache Tomcat 8.5+
- **数据库**：MySQL 5.7+
- **IDE**：IntelliJ IDEA (推荐)

## 5. 启动步骤

### 5.1 数据库准备
1. 启动MySQL服务
2. 创建数据库：`CREATE DATABASE shinebookshop;`
3. 导入数据：`source shinebookshop.sql`

### 5.2 项目启动
1. 配置Tomcat服务器
2. 部署项目到Tomcat
3. 启动Tomcat服务器
4. 访问：http://localhost:8080/shinebookshop

## 6. 注意事项
- 确保MySQL服务正在运行
- 检查数据库连接配置是否正确
- 确认Tomcat配置无误
- 检查防火墙设置

## 7. 常见问题
1. **数据库连接失败**：检查MySQL服务状态和连接参数
2. **编码问题**：确保数据库字符集为UTF-8
3. **端口冲突**：检查8080端口是否被占用
